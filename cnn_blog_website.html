Section 01

What is a CNN?

A CNN is a deep learning model designed to process structured grid data like images — preserving 2D structure rather than flattening it into a list of numbers.

Built for Images

Unlike regular neural networks, CNNs preserve spatial relationships. A cat's eye stays in the upper half of the image — and a CNN knows that matters.

Each layer learns increasingly abstract features: early layers detect edges and textures, while deeper layers recognise shapes, faces, and objects.

The result is a network that "sees" — the foundation of modern computer vision.

Section 03

The Architectures

Five models. Five breakthroughs. Each one asking — what did the last one get wrong?

1998
LeNet-5
Created by Yann LeCun at Bell Labs to read ZIP codes on postal mail. The first practical CNN — revolutionary for its time.
7 layers 60K params ~99% on MNIST
Key Innovations
Sigmoid & Tanh activations
Average pooling
Trained on MNIST (60K images)
2012
AlexNet
Won ImageNet 2012 with a 15.3% error rate vs runner-up's 26.2%. The moment deep learning became undeniable.
8 layers 60M params 15.3% error
Key Innovations
ReLU — 6× faster than Sigmoid
Dropout (50%) prevents overfitting
Dual-GPU training
2014
VGGNet
Oxford's simple answer: stack many small 3×3 filters instead of fewer large ones. Three 3×3 filters = same field as one 7×7, but with fewer params and more non-linearity.
16 layers 138M params 7.3% error
Key Innovations
Uniform 3×3 filter design
Extreme depth (up to 19 layers)
Gold standard for transfer learning
2014
GoogLeNet
Google's Inception module runs 1×1, 3×3, and 5×5 filters in parallel — letting the network decide which size works best at each point.
22 layers 5M params 6.67% error
Key Innovations
Inception modules (parallel convs)
27× fewer params than AlexNet
Global average pooling
2015
ResNet
Microsoft Research's brilliant insight: instead of learning H(x), learn just the residual F(x) = H(x) − x. Skip connections let gradients flow freely — enabling networks of 150+ layers.
152 layers 60M params 5.71% error
Key Innovations
Skip / residual connections
Solved vanishing gradient completely
Beat human-level accuracy (5.1%)

Section 06

Key Concepts

A quick-reference glossary of the essential terms you need to understand CNN literature.

CNN
A neural network designed for processing image data, preserving 2D spatial structure.
Convolution
A small filter slides over an image detecting features like edges and textures.
Pooling
Reducing feature map size while keeping the most important information.
ReLU
Activation function f(x) = max(0,x) — solves vanishing gradient, 6× faster training.
Dropout
Randomly switching off neurons during training to prevent overfitting.
Vanishing Gradient
Gradients become too small to update early layers — deep networks grind to a halt.
Skip Connection
A shortcut that lets gradients flow directly across layers — ResNet's core idea.
Transfer Learning
Using a pre-trained CNN (like ResNet) as the starting point for a new task.
Overfitting
Model memorises training data but fails on new, unseen examples.
Feature Map
The output produced after applying a convolutional filter to an image.
Filter / Kernel
A small matrix (e.g. 3×3) that detects specific patterns across an image.
Inception Module
Runs 1×1, 3×3, and 5×5 filters in parallel, letting the network pick the best.
